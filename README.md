# 古风小生 · Gu Feng Xiao Sheng

> 一位饱读诗书的文人，借诗、词、曲、经、楚辞之名句，以重度文言语风与人对谈。

`gu-feng-xiao-sheng` 是一款专为 [Oh My Pi (omp)](https://omp.sh/) 等支持 `SKILL.md` 加载机制的 AI 编程助手设计的 **managed skill**。它赋予 AI 助手一种「古风小生」人格：对话时以文言、章回口吻作答，并依当下话题意境，自然化用中国古典诗、词、曲、经、楚辞中的名句——恰似一位从线装书里走出来的书生，与你促膝而谈。

---

## 特性

- **重度古言语风**：文白相杂、章回式语句总领转折，再以白话释意；非堆砌，言简意赅为先。
- **场景化用**：从对方话语中抽取主题意境，自动匹配 18 个场景类目（离别送别、思乡怀人、四时春日……），援引贴切名句 0–2 联。
- **五大体裁**：诗（唐诗三百首）、词（宋词三百首）、曲（元曲名家）、经（诗经）、辞（楚辞），多体裁均衡配比。
- **触发词开关**：默认关闭，用户说「古风模式」「启古风」等触发词方才启用，言「恢复正常」即退。庸常事务（命令、路径、代码）不强行引诗，事实优先。
- **引必有据**：凡所引诗句，皆出于同目录 `scene_index.json`，不杜撰、不伪作。

## 快速开始

### 1. 安装

将本仓库克隆至 omp 的 managed-skills 目录：

```bash
git clone https://github.com/wuanshixue/gu-feng-xiao-sheng.git \
  ~/.omp/agent/managed-skills/gu-feng-xiao-sheng
```

目录结构应为：

```
~/.omp/agent/managed-skills/gu-feng-xiao-sheng/
├── SKILL.md          # 人格定义与引用规则
├── scene_index.json  # 305 条场景化诗句索引（18 场景类目）
├── README.md
└── LICENSE
```

下次启动 omp 会话时，技能即被自动加载至技能列表。

### 2. 激活与退出

本技能**默认关闭**，须以触发词开关：

| 状态 | 触发词（任一即生效） |
|---|---|
| **开启** | `古风模式`、`启古风`、`小生起`、`请用古风`、`化作古风小生`、`说人话我来听古风` |
| **关闭** | `关闭古风`、`复原`、`恢复正常`、`正常对话`、`不用古风了`、`止矣` |

- **开启**后，小生将以「古风已启，小生候命」类语回应，其后各轮均以古风口吻作答。
- **关闭**后，即恢复现代白话作答，亦有「古风已退」一语示之。
- 既未启又未关，则维持当前状态（默认为关闭）。

### 3. 效果示例

> **用户**：我的同事下个月就要调到外地分部了，我们共事五年了，心里很不是滋味。
>
> **古风小生**：话说君今为同僚远调之事，五载并肩，忽成分飞，这番怅惘，古人亦多有之。
>
> 杜牧《贈別》诗云：「蠟燭有心還惜別，替人垂淚到天明。」
>
> 其辞款曲——蜡尚有心而惜别，况于人乎？君与同僚共事五载，情谊自非泛泛，今虽各赴一方，留心相惜，亦复何伤。且以白话明之：既知其人将远，不妨趁这月余之时，把要说的话说透，把要做的事办妥，则虽别无憾矣。愿君察察。

## 工作原理

```mermaid
flowchart LR
  A[用户言语] --> B{含触发词?}
  B -->|否| C[现代白话作答]
  B -->|是| D[读取 scene_index.json]
  D --> E[抽取主题意境]
  E --> F[匹配 1-2 个场景类目]
  F --> G[择贴切诗句 0-2 联]
  G --> H[章回口吻 + 白话释意]
  H --> I[古风小生作答]
```

### 诗句索引格式

`scene_index.json` 以中文场景类目为键，值为诗句条目数组：

```json
{
  "离别送别": [
    {"f": "诗", "a": "杜牧", "t": "贈別二首", "q": "多情却似總無情……"},
    {"f": "词", "a": "李清照", "t": "凤凰台上忆吹箫", "q": "香冷金猊……"}
  ],
  ...
}
```

| 字段 | 含义 |
|---|---|
| `f` | 体裁：`诗` / `词` / `曲` / `经` / `辞` |
| `a` | 作者 |
| `t` | 标题（词多为词牌名） |
| `q` | 诗句成句（用于化用引用） |

共 305 条精选条目，每场景 7–19 条，体裁配额均衡。

## 自定义

- **增减触发词**：编辑 `SKILL.md` 第「零、激活开关」一节的表格。
- **增减场景类目**：编辑 `SKILL.md` 第「二、场景匹配法」一节的表格；同时在 `scene_index.json` 中增删对应键。
- **替换诗句**：直接编辑 `scene_index.json` 中相应场景下的条目即可；亦可自行重新构建索引（脚本见下文「致谢」节链接的源仓库）。
- **调节古风浓度**：若欲由「重度」降为「中度」，可将 `SKILL.md` 第一章「语风重度古化」改为「轻度古化」，减少章回套语的使用频率。
